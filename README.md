# 🌅 Sundown Studio — Frontend Clone

## 📌 Overview
A frontend clone of the Sundown Studio website built using HTML, CSS and 
JavaScript. Focused on modern UI interactions and smooth user experience.

## ✨ Features
- Custom orange cursor replacing default browser cursor
- Smooth scrolling using Locomotive.js
- Dynamic image popup on project hover
- Animated running text marquee
- Hover effects on navigation and project list
- Responsive animated blob shapes using CSS keyframes

## 🛠️ Tech Stack
HTML | CSS | JavaScript | Locomotive.js | GSAP

## 📁 Project Structure
- `Index.html` — Main page structure
- `style.css` — All styling and animations
- `script.js` — JavaScript interactions

## 🎯 What I Learned
- CSS animations and keyframes
- JavaScript DOM manipulation
- Third party library integration (Locomotive.js, GSAP)
- Modern UI/UX design patterns
